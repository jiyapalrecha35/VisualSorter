Explore Sorting Algorithms:

  • 🔄 Bubble SortO(n2) - Repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.
  • 🔀✅ Quick SortO(n2) - O(n log n) - Divides the list into smaller sublists and sorts them recursively.
  • 🎯 Selection SortO(n2) - Selects the smallest element from the unsorted portion and swaps it with the first element.
  • 🔄➡️ Merge SortO(n log n) - Divides the list into halves, sorts each half, and then merges them back together.
  • ➕ Insertion SortO(n2) - Builds the final sorted array one item at a time, inserting each element into its correct position.
  • 📚🔃 Heap SortO(n log n) - Converts the list into a binary heap and extracts the maximum element repeatedly to build the sorted array.
